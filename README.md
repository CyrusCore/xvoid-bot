# **xvoid-bot : a oriented bot for whatsapp**

# How To Install?
 
 ```js
> update && apt upgrade
> apt install git -y
> apt install nodejs -y
> apt install ffmpeg -y
> git clone https://github.com/CyrusCore/xvoid-bot
> cd xvoid-bot
> npm install 
> node . [session-name] 

```

Edit The Config On Config/Config.json and Config/api.json
